# Customer Churn Analysis
## Overview
Customer churn is one of the most critical business problems facing telecom companies today. Acquiring a new customer costs 5x more than retaining an existing one - making churn prediction and prevention a top business priority. This project analyzes 7,043 telecom customers across 21 features including contract type, internet service type, tenure, monthly charges and payment methods to uncover the root causes of customer churn. Using Python for data cleaning and exploratory analysis, and Tableau for visualization, this project identifies the highest risk customer segments and provides data-driven recommendations to reduce the company's 26.5% churn rate - potentially saving millions in lost revenue annually.
